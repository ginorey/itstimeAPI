    Introduction

    Welcome to the itsTIME API. You can use our API to access itsTIME API endpoints, which can get information on fighters in the UFC.

    This is an open source project. Feel free to open issues and pull requests.

    As this is an ongoing project, everything is subject to change. Currently you are able to pull only information from the fighters, but I will be adding the ability to pull information on each event.

    You can use my free API to access all UFC related data, no email or API key required.

    This API was created by me, Gino Rey. Feel free to message me on LinkedIn, or Discord if you have any questions!.

    Getting Started

    You need a computer with an internet connection.

    • No email required.
    • No API key required.
    • Contains data from all fighters in the entire UFC.
    • The data is updated once a day at 6:00AM EST.
    • Rate limit of 30 requests per minute.

    Errors

    The itsTIME API uses the following error

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    404 Not Found -- The specified data could not be found.
    406 Not Acceptable -- You requested a format that isn't json.
    429 Too Many Requests -- Slow down cowboy!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
